Desarrollo web Introducción El desarrollo web es una excelente entrada al mundo del desarrollo ya que cuenta con una extensa demanda y se encuentra siempre evolucionando. Toda introducción al desarrollo web debe empezar con HTML, CSS y JS, pues estas tecnologías conforman los pilares del desarrollo web front-end. Navegadores Actualmente en el mercado existen tres navegadores web, a continuación se enlistan según su popularidad. Chrome Safari Firefox Recursos adicionales Los sitios web más recomendados para aprender desarrollo web front-end: https://www.freecodecamp.org/ https://developer.mozilla.org/es/ https://www.codecademy.com/catalog/subject/web-development Glosario front-end: Se conoce como front-end a la rama de desarrollo que se encarga de diseñar y programar el aspecto visible de una aplicación, es con esta capa con la que el usuario interacciona. herramientas para desarrolladores: Un conjunto de herramientas que proveen funciones de depuración y análisis para los desarrolladores.
